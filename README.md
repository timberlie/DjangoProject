# DjangoProject

# 🎌 Веселье с флагами

Веб-приложение для изучения флагов мира в игровой форме.

## Автор
Тимберли

## Описание проекта

Проект представляет собой образовательный веб-сервис для изучения флагов разных стран. Приложение включает:

- Бесконечную игру "Угадай флаг" с тремя вариантами ответов
- Блог с интересными фактами о флагах
- Регистрацию и авторизацию пользователей
- Систему подсчёта очков

## Технологии

- Python 3.8
- Django 4.2
- SQLite
- HTML/CSS (собственные стили)
- JavaScript (AJAX)
